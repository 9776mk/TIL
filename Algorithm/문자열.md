# 2. 문자열 메서드

1. .split(기준 문자)
- 문자열을 일정 기준으로 나누어 리스트로 반환
- 괄호 안에 아무것도 넣지 않으면 자동으로 공백을 기준으로 설정
  ? 복잡도 : O(N) ?
2. .strip(제거할 문자)
- 문자열의 양쪽 끝에 있는 특정 문자를 모두 제거한 새로운 문자열 반환
- 괄호 안에 아무것도 넣지 않으면 자동으로 공백을 제거 문자로 설정
- 제거할 문자를 여러 개 넣으면 해당하는 모든 문자들을 제거
  ? 복잡도 : O(1) ?
3. .find(찾는 문자)
- 특정 문자가 처음으로 나타나는 위치(인덱스)를 반환
- 찾는  문자가 없다면 -1을 반환
  ? 복잡도 : O(N) ?
4. .index(찾는 문자)
- 특정 문자가 처음으로 나타나는 위치(인덱스)를 반환
- 찾는  문자가 없다면 오류 발생
  ? 복잡도 : O(N) ?
5. .count(개수를 셀 문자)
- 문자열에서 특정 문자가 몇 개인지 반환
- 문자 뿐만 아니라, 문자열의 개수도 확인 가능
  ? 복잡도 : O(N) ?
6. .replace(기존 문자, 새로운 문자)
- 문자열에서 기존 문자를 새로운 문자로 수정한 새로운 문자열 반환
- 특정 문자를 빈 문자열("")로 수정하여 마치 해당 문자를 삭제한 것 같은 효과 가능
  ? 복잡도 : O(N^2) ?
7. 삽입할 문자.join(iterable)
- iterable의 각각 원소 사이에 특정 문자를 삽입한 새로운 문자열 반환
- 공백 출력, 콤마 출력 등 원하는 형태를 위해 사용
  ? 복잡도 : O(N) ?

# 3. 아스키(ASCII) 코드

- 컴퓨터는 숫자만 이해할 수 있다!
  
  - 비트(bit) : 0과 1 두가지 정보만 표현
  - 바이트(byte) : 데이터를 저장하는 기본 단위 1 byte == 8bits

- 문자는 어떻게 저장될까?

- ASCII(American Standard Code for Information Interchange) : 미국 정보교환 표준부호
  
  - 알파벳을 표현하는 대표 인코딩 방식
  - 각 문자를 표현하는데 1byte(8bits) 사용
    - 1 bit : 통신 에러 검출용
    - 7 bit : 문자 정보 저장(총 128개)
  1. ord(문자)
  - 문자를 아스키 코드로 출력하는 내부 함수
  2. chr(아스키 코드)
  - 아스키 코드를 문자로 출력하는 내부 함수
